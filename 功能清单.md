# AI平台余额快捷入口 - 功能清单

## 项目概述

这是一个基于 React 18 + TypeScript + Vite + Tailwind CSS 的本地网页应用，用于集中管理和快速访问多个 AI 平台的余额页面。

---

## ✅ 已实现功能

### 1. 核心功能
- [x] **平台卡片展示** - 以网格或列表形式展示各 AI 平台
- [x] **快捷跳转** - 点击按钮在新标签页打开对应平台页面
- [x] **链接自定义** - 用户可自定义/修改各平台的跳转链接
- [x] **本地存储** - 使用 localStorage 保存用户自定义配置
- [x] **预设平台** - 内置 20+ 个常用 AI 平台配置

### 2. 平台管理
- [x] **添加新平台** - 用户可自定义添加新的平台卡片
- [x] **编辑平台** - 修改平台名称、链接、分类、图标等信息
- [x] **删除平台** - 删除不需要的平台卡片
- [x] **收藏功能** - 收藏常用平台，支持按收藏筛选
- [x] **重置配置** - 一键恢复到默认平台配置

### 3. 筛选与搜索
- [x] **搜索功能** - 按名称快速搜索平台
- [x] **分类筛选** - 按分类筛选（全部、云服务商、C端平台、其他、收藏）
- [x] **视图切换** - 网格视图 / 列表视图切换
- [x] **键盘快捷键** - Ctrl/Cmd + K 聚焦搜索，Esc 关闭弹窗

### 4. 批量操作
- [x] **批量选择** - 多选平台卡片
- [x] **批量打开** - 在新标签页批量打开选中的平台
- [x] **全选/取消全选** - 快速选择所有可见平台

### 5. 数据管理
- [x] **配置导出** - 将平台配置导出为 JSON 文件
- [x] **配置导入** - 从 JSON 文件导入平台配置
- [x] **数据持久化** - 所有配置自动保存到 localStorage

### 6. UI/UX
- [x] **主题切换** - 浅色 / 深色主题切换
- [x] **玻璃拟态风格** - 现代化的玻璃拟态 UI 设计
- [x] **落地页** - 首次访问时的欢迎引导页
- [x] **Toast 提示** - 操作成功/失败的提示消息
- [x] **空状态设计** - 友好的空状态引导
- [x] **视觉特效** - 粒子动画背景效果
- [x] **诗歌展示** - 随机展示一句古诗作为点缀
- [x] **响应式布局** - 适配不同屏幕尺寸
- [x] **无障碍支持** - aria-label 等无障碍属性

### 7. 预设平台（26个）
- 火山引擎、PromptPilot、DeepSeek、DeepSeek Chat
- MiniMax Agent、智谱GLM、月之暗面、智谱
- 阿里百炼、魔搭社区、Minimax、硅基流动（中国/国际）
- OpenRouter、AIHubMix、智谱开放平台、智谱国际
- AIGC Panel、Grok、ChatGPT、Canirun
- Manus、AI工具集、AI日报、Hugging Face
